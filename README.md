# FireFighter-Divers

აკვალანგის გარეშე არ გამოცხადდეთ!!!




1) შევქმენით პროექტის საწყისი ვერსია.

2) თქვენი სექციები გამოყოფილია html.index ფაილში
და შესაბამისი სექციებისთვის განკუთვნილი scss ფაილები ჩაყრილია syles-ფოლდერში.

3) მუშაობის დაწყებამდე ვხსნით ტერმინალს (styles-ფოლდერში უნდა ვიდგეთ) და ვწერთ შემდეგ ბრძანეას 
                           sass --watch style.scss style.css

 გაფრთხილება!!! იმუშავეთ იმ scss ფაილებში რომელიც გეკუთვნით მხოლოდ თქვენ!

4) ვაკეთებთ საკუთარ branch-ს სადაც ვიმუშავებთ კოდის ჩვენივე ნაწილზე

5) დავალების შესრულების პროცესში თქვენთვითონ უნდა შეასრულოთ თქვენი სექციის responsive თქვენსავე scss - ფაილში

6) ყურადღება მიაქციეთ dark-light mode-ს იმ შემთხვევაში თუ იმ სექციაზე რომელზეც თქვენ მუშაობთ რომელიმე ელემენტს dark-mode-ში ფერი ეცვლება ამ ელემენტს აუცილებლად დაუწერეთ კლასის სახელი "switch-theme" 

      მაგალითისთვის >> <h1 class="switch-theme">ეს ფერს შეიცვლის თავისით</h1>

ეს საშუალებას მოცემთ რომ თემის შეცვლის დროს თქვენი ელემენტის სტილი მოერგოს dark-mode-ს (შესაბამისად ამისთვის უკვე არის გამზადებული კოდი ჯავასკრიფტ ფაილში)

გაფრთხილებაა!!!  switch-theme ამ კლასზე scss-ში სტილებს და ასშ... რაფერს არ ვწერთ
თუ გასტილვა გვინდა აუცილებელია ელემენტს საკუთარი კლასი მივუთითოთ და ის გავსტილოთ
      მაგალითისთვის >> <h1 class="myclass switch-theme">ჩვენი სტილები იმოქმედებს და ფერს შეიცვლის თავისით</h1>

ასევე თუ გაქვთ აიქონები რომელიც იცვლის ფერს ამ შემთხვევაში კლასი იქნება light-icon და dark-icon ორივე შემოიტანეთ და შავ აიქონებს რომელიც თეთრ თემაზე ჯდება დაარქვით light-icon და რომელიც დარკ თემაში უნდა იჯდეს დაარქვით dark-icon


7) ასევე თუ ნახავთ abstract-ფოლდერს შიგნით არის მოთავსებული ფაილი default.scss სადაც არის ღილაკებზე უკვე გამზადებული სტილები და თუ გვინდა რომ ეს შემოვიტანოთ ჩვენ scss ფაილში ვაკეთებთ შემდეგ ნაირად 

.myelement {
    <!-- ცვლადის მაგალითი სადაც მხოლოდ ერთი სტიილი გვაქვს მოთავსებული -->
    color: default.$primary-color; 
    <!-- მიქსინის მაგალითი სადაც ბევრი სტილი გვაქვს მოთავსებული -->
    @include default.primary-btn-mobile
}

default ნიშნავს რომ ამის მიქსინი ან ცვლადი მოთავსებულია default.scss ფაილში და ამის მისამართი დაინპორტებულია თქვენსავე სამუშაო ფაილის სულ პირველ ხაზში ასერომ მას არ ვეხებით ^_^ <3
